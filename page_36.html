Brand Variables

These CSS custom properties (`:root` variables) define the core values for our brand's identity, color palette, typography, and spacing. By centralizing these, we ensure consistency and easy updates across all projects. 


:root {
  /* Brand identity */
  --brand-name: "Style Guide App";

  /* Color palette */
  --color-primary-blue: #3b82f6;
  --color-secondary-blue: #2563eb;
  --color-background: #f5f5f4;
  --color-surface: #ffffff;
  --color-text-dark: #292524;
  --color-text-medium: #78716c;
  --color-text-light: #a8a29e;
  --color-border-default: #e7e5e4;

  /* Typography */
  --font-family-base: 'Inter', sans-serif;
  --font-size-base: 16px;
  --line-height-base: 1.6;
  --font-size-h1: 2.5rem;
  --font-size-h2: 2rem;

  /* Spacing scale (multiples of 4px) */
  --space-0: 0;
  --space-1: 4px;
  --space-2: 8px;
  --space-3: 16px;
  --space-4: 24px;
  --space-5: 32px;

  /* Border & radius */
  --border-radius-sm: 0.25rem;
  --border-radius-md: 0.5rem;
  --border-width-default: 1px;
}

Responsive Design

Example of media queries for responsive adjustments across different screen sizes. Watch the box change color and padding as you resize the window.

Resize me!

Current color/padding changes based on breakpoint


/* Mobile First styles are default */
.responsive-box {
  background-color: var(--color-primary-blue);
  color: white;
  padding: var(--space-2); /* 8px */
}

/* Small devices (sm: 640px and up) */
@media (min-width: 640px) {
  .responsive-box {
    background-color: var(--color-secondary-blue);
    padding: var(--space-3); /* 16px */
  }
}

/* Medium devices (md: 768px and up) */
@media (min-width: 768px) {
  .responsive-box {
    background-color: #05DBF2; /* A lighter blue */
    padding: var(--space-4); /* 24px */
  }
}

/* Large devices (lg: 1024px and up) */
@media (min-width: 1024px) {
  .responsive-box {
    background-color: #004AAD; /* A darker blue */
    padding: var(--space-5); /* 32px */
  }
}
